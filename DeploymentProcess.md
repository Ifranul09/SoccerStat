Create a database called SoccerStat in your mysql command prompt. <br>
Download the SoccerDatabase.sql file from this github. <br>
In the regular command prompt, input the following: mysql -u [your username] -p SoccerStat < SoccerDatabase.sql <br>
Enter your own mysql password when prompted. <br>
Log back into the mysql command prompt and use the SoccerStat database. <br>
